We first begin by splitting our data set from the first 20 years (1977 - 1997) into a training set and a testing set. From that, we create a random forest tree model which we then make sure is functional by testing it against the testing set for the first 20 years. The reason we are using the first 20 years as our model training and testing set is because we believe that no one who retired more than 20 years ago is going to be moved into the Hall of Fame now so everyone on that list's position either in the HoF or not are solidified. After testing the accuracy of the model, we run the model against players who are either still playing or who have retired in the more recent 20 years in order to see who belongs in the HoF today
